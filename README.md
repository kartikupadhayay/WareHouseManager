WareHouse Manager is a desktop application developed to manage the inventory and delivery operations of a warehouse. 
It provides warehouse operators with a clean, simple, and fully functional graphical interface that allows them to manage their entire inventory without any knowledge of databases, SQL, or programming.
Every operation from adding a new item to the warehouse, to shipping goods out for delivery is just a button click away.

The application is built using Python, one of the most widely used programming languages in the world, with Tkinter as the graphical user interface framework.
The data is stored and managed on Snowflake, a powerful cloud-based relational database platform, ensuring that the data is always available, secure, and consistent.
The two components are connected using the official Snowflake Python connector, which allows the application to execute live database queries in real time.

This project was developed as part of a Database Management Systems course, with the goal of demonstrating practical application of core DBMS concepts including table design, SQL queries, data integrity, and real-time database connectivity within a working software application.
